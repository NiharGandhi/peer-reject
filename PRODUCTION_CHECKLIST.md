# Production Checklist — peer-reject

Generated by Deployfast. Work through this before going live.

## Security
- [ ] All secrets in environment variables, none hardcoded
- [ ] `.env` is in `.gitignore`
- [ ] Dependencies are up to date (`npm audit` / `pip audit`)
- [ ] CORS configured to allow only your frontend domain
- [ ] Rate limiting enabled on API routes
- [ ] HTTPS enforced in production

## Database
- [ ] Migrations run on deploy, not manually
- [ ] Database backups configured
- [ ] Connection pooling enabled
- [ ] Indexes on frequently queried columns

## Docker
- [ ] Multi-stage build (reduces image size)
- [ ] Non-root user in container
- [ ] Health check defined
- [ ] `.dockerignore` excludes `node_modules`, `.env`, `.git`

## CI/CD
- [ ] Tests run on every PR
- [ ] Linting enforced
- [ ] Docker image builds on every push to main
- [ ] Deployment is automatic on merge to main

Add this file to your repo at `.github/workflows/ci.yml`:

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'npm'

      - name: Install dependencies
        run: npm ci

      - name: Lint
        run: npm run lint

      - name: Build
        run: npm run build

      - name: Build Docker image
        run: docker build -t app .
```

## Observability
- [ ] Structured logging (JSON format)
- [ ] Error tracking (Sentry or equivalent)
- [ ] Uptime monitoring
- [ ] Basic metrics (response time, error rate)

## Performance
- [ ] Static pages use SSG where possible
- [ ] Images use next/image
- [ ] Bundle size analyzed (`next build` output)
- [ ] Response caching where appropriate
- [ ] Assets served from CDN

## Before Launch
- [ ] Custom domain configured
- [ ] SSL certificate active
- [ ] Environment variables set in production
- [ ] Smoke test on production URL
- [ ] Rollback plan documented
